Principais Formatações

Negrito / Destaque

Para deixar a frase com um termo em negrito usamos a tag <b>.
Para deixar a frase com um termo em destaque usamos a tag <strong>.

Itálico/Ênfase

Podemos deixar um texto em italico com a tag <i>
Mas para deixa-lo com ênfase (semantica) usamos a tag <em>

Outras formatações que poderão ser úteis

Texto marcado

Para marcar um texto usamos a tag <mark>

Texto pequeno

Temos a tag <small> que serve para deixar um texto pequeno como os textos de rodapé de um contrato.

Texto deletado

Para desconsiderar um texto podemos utilizar a tag <del> que riscar o texto indicado.

Texto inserido

Para marcar um texto como inserido usamos a tag <ins> que sublinha o texto.
Temos tambem o sublinhado com a tag <u> (não semântico)

Texto sobrescrito e subscrito.

Para inserir do tipo x20+3 usamos a tag <sup>
e para inserir coisas como H 2O usamos a tag <sub>
